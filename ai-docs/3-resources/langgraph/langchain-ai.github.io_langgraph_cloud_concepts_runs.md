# Runs[¶](https://langchain-ai.github.io/langgraph/cloud/concepts/runs/#runs "Permanent link")
A run is an invocation of an [assistant](https://langchain-ai.github.io/langgraph/concepts/assistants/). Each run may have its own input, configuration, and metadata, which may affect execution and output of the underlying graph. A run can optionally be executed on a [thread](https://langchain-ai.github.io/langgraph/cloud/concepts/threads/).
The LangGraph Cloud API provides several endpoints for creating and managing runs. See the [API reference](https://langchain-ai.github.io/langgraph/cloud/reference/api/api_ref.html#tag/thread-runs/) for more details.
